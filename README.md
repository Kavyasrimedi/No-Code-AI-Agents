# No-Code-AI-Agents
A curated set of no-code AI agents designed on Agent.ai. This repo documents the prompts, workflows, reasoning logic, test inputs, and example outputs for each agent, for quick reuse and evaluation.

## Project Index

- `legal-document-analyzer/` — Legal Document Analyzer (main hackathon submission).
- `micro-study-planner/` — Micro Study Planner (student-focused agent).
- `job-application-tailor/` — Job Application Tailor (resume & cover letter assistant).

## How to use
Each project folder contains:
- `README.md` — project overview and instructions.
- `prompt.txt` — exact LLM prompt used in Agent.ai.
- `workflow.png` — screenshot of Agent.ai workflow (upload from UI).
- `sample-output.html` — example of rendered HTML output.
